# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added

- Initial release
- Move files between directories based on extension filters
- JSON configuration file support (`.tidyrc.json`)
- Environment variable configuration
- CLI arguments support
- Duplicate handling strategies: rename, skip, overwrite
- Dry-run mode for previewing changes
- Colorful terminal output with TTY detection
- Comprehensive exclusion patterns (files and globs)

## v1.1.0 (2026-01-20)

### Feat

- implement setuptools-scm for automatic git-based versioning

## v1.0.0 (2026-01-20)

### Feat
